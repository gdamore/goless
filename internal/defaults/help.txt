Navigation
Up/Down, j/k: move
Left/Right, h/l: scroll horizontally
Ctrl-U/Ctrl-D, u/d: half-page
PgUp/PgDn, b/f, Space: page
g/G: top/bottom
w: toggle wrap

Search
/: search forward
?: search backward
n/N: next/previous match
F2: cycle search case mode (smart/case/nocase)
F3: cycle substring/whole-word/regex matching

Commands
:123: jump to line 123
:50%: jump to 50%
:set numbers on|off|toggle
:set headers on|off|toggle|<n>
:set headercols on|off|toggle|<n>
:set searchcase smart|case|nocase
:set searchmode sub|word|regex

General
H or F1: help
q or Esc: quit
