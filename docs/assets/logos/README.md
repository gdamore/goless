# Logos

Place project logo source files and exported web-ready assets here.

Intended use:

- source exports and working logo variants kept in the repository
- assets copied into `site/static/assets/logos/` for the Hugo site
- README or site references served as `/assets/logos/...`

For Hugo and GitHub Pages, commit served assets into `site/static/assets/logos/`.
Within site content, reference them with absolute public paths such as
`/assets/logos/goless-light-transparent.png`.

Suggested contents:

- original PNG exports provided by design
- optimized variants for web if needed
- light/dark or icon/wordmark variants
